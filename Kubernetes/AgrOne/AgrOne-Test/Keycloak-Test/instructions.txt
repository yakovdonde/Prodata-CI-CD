This helm chart values assume:
- Keycloak url is: https://keycloak-prod.prodata.az
- Keycloak Admin url is: https://keycloak-prod.prodata.az
- Keycloak admin credentials are: admin / admin


In order to deploy keycloack with this helm chart run:
helm install keycloak-prod oci://registry-1.docker.io/bitnamicharts/keycloak -n agrotech-prod -f /kubernetes/agrotech-prod/keycloak/values.yaml
